5️⃣ LANCER L'EXTENSION
   ┌─────────────────────────────────────────────────────────┐
   │ Cliquez sur l'icône "LA"                                 │
   │ Cliquez "Démarrer"                                       │
   └─────────────────────────────────────────────────────────┘

6️⃣ VÉRIFIER LES LOGS (Console F12)
   ┌─────────────────────────────────────────────────────────┐
   │ Vous devriez voir:                                       │

   │                                                          │
   │ Si vous voyez:                                           │
   │ ❌ "Aucune offre trouvée"                               │
