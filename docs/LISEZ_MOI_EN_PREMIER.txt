✅ L'extension est PRÊTE avec les icônes installées!

📦 INSTALLATION (3 étapes simples):
──────────────────────────────────────────────────────────────

1. Ouvrez Chrome et tapez: chrome://extensions/

2. Activez "Mode développeur" (bouton en haut à droite)

3. Cliquez "Charger l'extension non empaquetée"
   → Sélectionnez ce dossier: C:\ExtensionAutojob

4. ✅ Une icône bleue "LA" apparaît dans Chrome!


⚙️ CONFIGURATION (30 secondes):
──────────────────────────────────────────────────────────────

1. Cliquez sur l'icône bleue "LA" dans Chrome

2. Remplissez vos informations personnelles
   (déjà pré-remplies avec vos données)

3. Cliquez "Sauvegarder la configuration"


🧪 TEST RAPIDE:
──────────────────────────────────────────────────────────────

Ouvrez ce fichier dans Chrome:
file:///C:/ExtensionAutojob/test/test.html

Cliquez sur l'extension → "Démarrer"
Les formulaires doivent se remplir automatiquement!
